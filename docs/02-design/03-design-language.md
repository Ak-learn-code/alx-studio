# Design Language

Version: 1.0  
Status: Active

## Direction

Dark Streetwear Editorial.

## Keywords

- schwarz
- silber
- clean
- urban
- modern
- präzise
- reduziert
- hochwertig
- großflächig
- rhythmisch

## Visual Rules

- Dark Mode only
- große Typografie
- viel Abstand
- klare Raster
- wenige Akzentfarben
- keine übertriebenen Neon-Effekte
- keine generische AI-Ästhetik

## Feeling

Die Seite soll wirken wie ein Mix aus Creative Studio, digitalem Magazin und moderner Streetwear-Marke.
