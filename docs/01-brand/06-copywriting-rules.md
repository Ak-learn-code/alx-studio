# Copywriting Rules

Version: 1.0  
Status: Active

## Rules

Kurz schreiben.  
Direkt schreiben.  
Keine Buzzwords.  
Keine künstliche Größe.  
Persönlich bleiben.  
Junior-Status nicht verstecken, sondern als Entwicklung zeigen.

## Preferred Words

Ideen, Systeme, entwickeln, verbessern, ausprobieren, digital, modern, klar.
