# Motion System

Version: 1.0  
Status: Active

## Philosophy

Motion erzeugt Fokus, Orientierung und Qualität.

## Durations

Micro: 150ms–250ms  
Default: 300ms–500ms  
Large: 600ms–800ms  

## Easing

Smooth ease-out. Keine hektischen Bewegungen.

## Allowed

- Scroll reveal
- Hover lift
- subtle parallax
- staggered text
- smooth section transitions

## Avoid

- endlose Loops
- wilde 3D-Effekte
- blinkende Elemente
