# Scroll Animations

Version: 1.0  
Status: Active

## Rules

Sections erscheinen beim Scrollen subtil.

## Pattern

Initial: opacity 0, y 32  
Animate: opacity 1, y 0  
Duration: 500ms  
Stagger: 80ms  

## Usage

Hero, About, Cards, Timeline, Contact.

## Avoid

Keine Scrollanimation darf den Content schwer lesbar machen.
